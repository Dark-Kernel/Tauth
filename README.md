# Tauth
### Termux fingerprint authentication 

***#This works only on the devices having fingerprint sensor.***

***#Make sure you have installed termux-api from playstore.***

#RUN :

	git clone https://github.com/Mr-SimpleEditor/Tauth
	
	cd Tauth 
	
	./setup
